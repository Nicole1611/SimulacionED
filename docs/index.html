    Cola (Queue)

    ¿Qué es?

    Una cola es una estructura FIFO donde el primer elemento en entrar es el primero en salir.

    Representación gráfica

    Tipos de colas

      Cola simple

      Estructura FIFO (First In, First Out) donde los elementos se insertan por el final y se eliminan por el inicio.

    Lista Enlazada (LinkedList)

    ¿Qué es?

    Una lista enlazada es una estructura compuesta por nodos donde cada nodo apunta al siguiente.
