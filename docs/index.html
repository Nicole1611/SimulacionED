    Cola (Queue)

    ¿Qué es?

    Una cola es una estructura FIFO donde el primer elemento en entrar es el primero en salir.

    Tipos de colas

      Cola simple

      Estructura FIFO (First In, First Out) donde los elementos se insertan por el final y se eliminan por el inicio.

    Árbol

    ¿Qué es?

    Un árbol es una estructura de datos jerárquica formada por nodos, donde cada nodo puede tener uno o varios hijos. Se organiza a partir de un nodo principal llamado raíz y permite representar relaciones padre-hijo.

    Tipos de árboles

    Árbol binario

    Estructura de árbol donde cada nodo tiene como máximo dos hijos: hijo izquierdo y hijo derecho. Es ampliamente usado en búsquedas, ordenamiento y evaluación de expresiones.

    Árbol general

    ¿Qué es?

    Un árbol general es una estructura de datos jerárquica formada por nodos, donde cada nodo puede tener cualquier número de hijos. Se organiza a partir de un nodo principal llamado raíz y permite representar relaciones complejas de tipo padre-hijo.

    Características

    • No existe un límite fijo en la cantidad de hijos por nodo.
    • Se utiliza para representar jerarquías como sistemas de archivos, organigramas o menús.
    • Los nodos se organizan en niveles.
